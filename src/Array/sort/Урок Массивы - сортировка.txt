    В Java и других языках программирования существует множество алгоритмов сортировки массивов, помимо сортировки
    пузырьком. Некоторые из наиболее распространенных видов сортировки, которые может быть полезно изучить, включают:

    1) Сортировка вставками (Insertion Sort): Алгоритм сортировки, который сравнивает элементы массива и вставляет их на
    правильные позиции в отсортированную часть массива.

    2) Сортировка выбором (Selection Sort): Алгоритм сортировки, который находит наименьший (или наибольший) элемент массива
    и перемещает его на соответствующую позицию в отсортированной части массива.

    3) Сортировка слиянием (Merge Sort): Алгоритм сортировки, который использует подход "разделяй и властвуй", разделяя
    массив на две половины, рекурсивно сортирующие их, а затем объединяющие отсортированные половины.

    4) Быстрая сортировка (Quick Sort): Алгоритм сортировки, который также использует подход "разделяй и властвуй", выбирая
    опорный элемент из массива, разделяя массив на две части относительно опорного элемента, и рекурсивно сортируя эти
    части.

    5) Сортировка подсчетом (Counting Sort): Алгоритм сортировки, который основывается на подсчете количества вхождений
    каждого элемента массива и размещении элементов в правильном порядке на основе этого подсчета.

    6) Сортировка пирамидальная (Heap Sort): Алгоритм сортировки, который строит двоичное дерево (кучу) из элементов
    массива и затем выполняет операции "просеивания" (sift) для превращения кучи в отсортированный массив.

    Это только некоторые из наиболее распространенных алгоритмов сортировки, и в языке Java также предоставляются
    различные встроенные методы сортировки, такие как Arrays.sort() и Collections.sort(), которые могут быть использованы
    для сортировки массивов и коллекций. Знание различных алгоритмов сортировки может быть полезно при выборе наиболее
    подходящего алгоритма для конкретной задачи, учитывая время выполнения, требования по памяти и другие факторы.
